# Pertemuan 1 - Pengantar Web Service & Sistem Terdistribusi

## Tujuan

Mahasiswa memahami posisi web service dalam sistem modern.

## Materi

- Konsep client-server
- Web service vs web application
- Evolusi komunikasi sistem terdistribusi
- Overview REST, RPC, dan event-driven

## Praktik

1. Buat diagram sederhana client-server untuk kasus perpustakaan.
2. Tandai mana bagian client, server, database, dan API.
3. Tulis minimal tiga contoh komunikasi:
   - REST untuk request data buku
   - RPC untuk pemanggilan fungsi antar service
   - event-driven untuk notifikasi peminjaman buku

## Output

Dokumen singkat berisi diagram dan penjelasan posisi web service.

Contoh hasil praktik tersedia di [docs/hasil/pertemuan-01.md](../hasil/pertemuan-01.md).
