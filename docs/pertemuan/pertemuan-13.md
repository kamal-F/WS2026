# Pertemuan 13 - RPC & gRPC

## Tujuan

Mahasiswa memahami komunikasi service-to-service berbasis RPC.

## Materi

- REST vs RPC
- Konsep `.proto`
- gRPC service communication

## Praktik

1. Buat file `.proto` untuk service `BookService`.
2. Definisikan method `GetBook`.
3. Jalankan server gRPC sederhana.
4. Buat client gRPC untuk memanggil service.
5. Bandingkan pengalaman call gRPC dengan REST endpoint.

## Output

Demo gRPC call antar service.
