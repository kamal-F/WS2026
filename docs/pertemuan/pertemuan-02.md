# Pertemuan 2 - HTTP & RESTful Principles

## Tujuan

Mahasiswa memahami prinsip REST secara benar.

## Materi

- HTTP methods
- HTTP status code
- Statelessness
- REST constraints

## Praktik

1. Jalankan API lokal.
2. Coba endpoint `GET /health`.
3. Catat contoh penggunaan method `GET`, `POST`, `PUT`, dan `DELETE`.
4. Buat tabel status code yang muncul dari endpoint sukses, validasi gagal, route tidak ditemukan, dan resource tidak ditemukan.

## Output

Tabel eksperimen HTTP method dan status code.

Contoh hasil praktik tersedia di [docs/hasil/pertemuan-02.md](../hasil/pertemuan-02.md).
