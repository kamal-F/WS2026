# Pertemuan 12 - Message Queue dengan RabbitMQ

## Tujuan

Microservice berbasis event.

## Materi

- Asynchronous communication
- Producer-consumer pattern
- Event-driven microservice

## Praktik

1. Jalankan RabbitMQ di localhost.
2. Buat producer event `book.created`.
3. Buat consumer yang membaca event dan menulis log.
4. Diskusikan apa yang terjadi jika consumer mati sementara producer tetap berjalan.

## Output

Publish dan consume event dengan RabbitMQ.
