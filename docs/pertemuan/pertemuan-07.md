# Pertemuan 7 - SOAP Web Service & WSDL

## Tujuan

Mahasiswa memahami perbedaan SOAP vs REST dan konsep service contract.

## Materi

- Konsep SOAP
- XML message structure
- Konsep WSDL
- Contract-first web service

## Praktik

1. Baca contoh struktur WSDL.
2. Identifikasi `operation`, `message`, `binding`, dan `service`.
3. Buat contoh SOAP request dan SOAP response untuk operasi `GetBook`.
4. Bandingkan kontrak SOAP dengan endpoint REST `GET /api/v1/books/:id`.

## Output

Catatan perbandingan SOAP contract-first dan REST resource-oriented.
