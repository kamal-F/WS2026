# Pertemuan 10 - Authentication & Security Dasar

## Tujuan

API memiliki keamanan dasar.

## Materi

- JWT authentication
- API key
- CORS dan rate limiting konsep

## Praktik

1. Buat endpoint login sederhana.
2. Generate JWT ketika login berhasil.
3. Lindungi endpoint write operation dengan middleware auth.
4. Bandingkan skenario JWT dan API key.
5. Catat risiko dasar: token bocor, token expired, dan CORS yang terlalu longgar.

## Output

REST API dengan auth JWT dasar.
